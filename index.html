drachtio-fs-load-balancing-proxy

A nodejs-based load-balancing sip proxy for Freeswitch

View on GitHub Download .zip Download .tar.gz

drachtio-fs-load-balancing-proxy is a nodejs-based SIP load balancer for Freeswitch servers. It is based on the high-performance drachtio signaling resource framework, which in turn utilizes the sofia SIP stack.

The need to load balance transactions across a horizontal cluster of Freeswitch servers is a common requirement for enterprises looking to scale their Freeswitch-based application environment. The most common solution to date has been the Kamailio SIP proxy, which provides a solution employing a proprietary script-driven configuration language. The solution presented here may be more desirable to nodejs developers as well as to developers seeking to build a wider range of applications than is possible with Kamailio.

drachtio is an open-source, nodejs-based ecosystem for creating any kind of VoIP server-based application: registrar, proxy, back-to-back user agent, and many others. Furthermore, when coupled with the drachtio media resource function, rich media-processing applications can be easily built as well.
